# PRD: IMEGuard

## 1. 개요

### 1.1 프로젝트 소개
IMEGuard는 Windows 환경에서 한/영 IME 관련 불편함을 해소하는 AutoHotkey 기반 유틸리티 프로그램이다.
터미널, 에디터 등 개발 작업 중 발생하는 한글 입력 오동작을 자동으로 처리하고,
현재 IME 상태를 시각적으로 알려주는 것을 목표로 한다.

### 1.2 배경 및 문제 정의

| 문제 | 상황 |
|------|------|
| ESC 연타 시 한/영 토글 | Vim 계열 에디터, 터미널에서 명령 모드 진입 시 한글일 경우 영문으로 바꿔줘야하는 번거로움. |
| 터미널 IME 포커스 오류 | 한/영 전환 키를 눌러도 영문만 입력됨. 메모장 등 다른 창을 거쳐야 한글 입력이 됨 |
| 현재 IME 상태 파악 어려움 | 영문인지 한글인지 인지하지 못한 채 입력하다 오류 발생 |

---

## 2. 목표

- ESC 키를 누르면 IME 상태와 무관하게 영문으로 고정
- 한/영 전환 시 현재 상태를 토스트 팝업으로 1.5초간 표시
- 터미널에서 한글 입력이 안 되는 IME 포커스 오류를 트레이 메뉴 클릭 한 번으로 복구
- 트레이 아이콘으로 기능 ON/OFF 제어
- 단일 EXE 배포 (설치 불필요)
- 트레이에서 시작프로그램 등록 기능 필요

---

## 3. 기능 요구사항

### 3.1 ESC → 영문 고정 (필수)

- ESC 키 입력 시 현재 IME 상태를 확인
- 한글 모드인 경우 영문으로 전환 후 ESC 이벤트 전달
- 영문 모드인 경우 ESC 이벤트만 전달 (토글 없음)
- 연타해도 영문 유지 (여러 번 눌러도 안전)
- 동작 범위: 시스템 전체

### 3.2 토스트 IME 상태 표시 (필수)

**표시 조건 (아래 상황에서만 표시)**

| 트리거 | 표시 내용 |
|--------|-----------|
| 한/영 키 입력 | 전환 후 상태 |
| ESC로 영문 전환 시 | EN |
| 트레이 복구 메뉴 클릭 시 | 복구된 상태 |

**표시 스펙**

- 위치: 활성 모니터 하단 중앙 고정
- 표시 시간: 1.5초
- 애니메이션: 페이드 인 (0.2초) → 유지 → 페이드 아웃 (0.5초)
- 항상 최상위 표시 (Always on Top)
- 클릭 통과 (Click-through, 마우스 이벤트 무시)

**디자인 스펙**

| 상태 | 표시 텍스트 | 색상 |
|------|-------------|------|
| 한글 | 한 | 붉은 계열 배경 + 흰 텍스트 |
| 영문 | EN | 파란 계열 배경 + 흰 텍스트 |

- 크기: 약 80×80px, 모서리 둥글게 (라운드 사각형)
- 반투명 처리 (Alpha 약 220/255)

### 3.3 IME 포커스 오류 복구 (필수)

**문제 상황**
터미널에서 한/영 키를 눌러도 영문만 입력되는 현상.
대상 터미널: Windows Terminal, WSL2, PowerShell, Warp

**해결 방식**

```
트레이 메뉴 "IME 포커스 복구" 클릭
→ 오프스크린 더미 윈도우로 포커스 이동
→ IME 한글 강제 활성화
→ 원래 창으로 포커스 복귀
→ 토스트 표시
```

- 동작 후 현재 창 포커스 유지
- 실패 시 트레이 알림 표시

### 3.4 시스템 트레이 (필수)

- 트레이 아이콘 상시 표시
- 우클릭 메뉴 구성:

```
IME 포커스 복구          ← 기본 항목 (더블클릭도 동작)
─────────────────
✅ ESC 영문 고정  (ON/OFF)
✅ 토스트 표시    (ON/OFF)
─────────────────
✅ 시작시 자동 실행 (ON/OFF)
─────────────────
설정...
─────────────────
종료
```

- 각 기능의 ON/OFF 상태는 ini 파일에 저장 (재시작 시 유지)
- "설정..." → `%AppData%\IMEGuard\IMEGuard.ini` 를 메모장으로 열기

## 4. 비기능 요구사항

| 항목 | 내용 |
|------|------|
| 배포 형태 | 단일 EXE (AHK 설치 불필요) |
| 시작 프로그램 | 최초 실행 시 시작 프로그램 등록 여부 선택 팝업, 트레이 아이콘에서 설정 |
| 설정 저장 | `%AppData%\IMEGuard\IMEGuard.ini` (EXE 위치 무관) |
| 메모리 | 상시 실행 기준 최소화 (트레이 상주) |
| OS 지원 | Windows 10 / 11 |
| AHK 버전 | AutoHotkey v2 |

---

## 5. 설정 파일 (IMEGuard.ini)

경로: `%AppData%\IMEGuard\IMEGuard.ini`

```ini
[Features]
EscForceEnglish=1
ToastEnabled=1

[Toast]
DisplaySeconds=1.5

[System]
RunAtStartup=1
```
